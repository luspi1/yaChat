# Веб-приложение "Чат"

Простой мессенджер со следующими возможностями:
-создание чата;
-изменение аватара чата;
-удаление чата;
-добавление и удаление пользователей из чата;
-поиск по всем чатам;
-отправка сообщений.

## Команды

Режим разработки
```sh
npm run dev
```

Сборка проекта
```sh
npm run build
```

Запуск проекта на сервере express js
```sh
npm run start
```
